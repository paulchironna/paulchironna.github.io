Rigol DS1054Z Repair Documentation

This project documents the diagnosis and repair of a Rigol DS1054Z digital oscilloscope that was displaying inaccurate and distorted waveforms on all input channels. Upon startup, the scope exhibited an attenuated and noisy waveform on CH1 even when a clean test signal was applied. Initial checks revealed an issue with the signal path near the input attenuator section. The following images and notes describe the fault symptoms, investigation process, and eventual restoration of proper function.

Initial waveform issue observed on CH1 using probe compensation output

Findings During Investigation

Further inspection of the input stage revealed multiple resistors in the attenuator network had drifted out of spec. The waveform distortion was traced to one of the 85Ω termination resistors connected to the BNC input. Using a working scope as a reference confirmed signal attenuation well before reaching the AFE.
